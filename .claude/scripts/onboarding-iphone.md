# Computer Onboarding — iPhone

> Step-by-step checklist for setting up a new iPhone for databayt work.
> Time: ~10 minutes.

## Prerequisites

- Apple ID signed in
- Wi-Fi connected
- iCloud Keychain enabled (Settings > Apple ID > iCloud > Passwords & Keychain)

---

## Apps to Install (App Store)

Open App Store and install each:

- [ ] **Claude** — AI assistant (search "Claude by Anthropic")
- [ ] **GitHub Mobile** — repos, issues, PRs, code review
- [ ] **Slack** — team communication (if workspace active)
- [ ] **Safari** — already installed, configure below

### Optional

- [ ] **Outlook** — company email (hi@databayt.org)
- [ ] **Notes** — dispatch/handoff with Claude Code (already installed)
- [ ] **Wispr Flow** — voice-to-text dictation

---

## Account Sign-ins

### Claude iOS
1. Open Claude app
2. Sign in with Anthropic account
3. Enable voice mode (microphone icon)
4. Verify conversations sync across devices

### GitHub Mobile
1. Open GitHub app
2. Sign in with GitHub account
3. Go to databayt org — verify you see repos (hogwarts, kun, etc.)
4. Enable notifications for mentions and review requests

### Slack (if active)
1. Open Slack
2. Join workspace: use invite link from team lead
3. Join channels: #general, #dev, #qa

---

## Safari Configuration

1. **Settings > Safari > AutoFill** — enable all (contact info, credit cards, passwords)
2. **Settings > Passwords** — verify iCloud Keychain has databayt service passwords
3. Test: open `github.com` in Safari — should autofill login

---

## Notifications

Configure for async team communication:

1. **Claude** — allow notifications (for shared conversations)
2. **GitHub** — allow notifications (for PR reviews, issue mentions)
3. **Slack** — allow notifications (for direct messages, mentions only)
4. **Notes** — allow notifications (for dispatch handoffs)

---

## Voice Mode Setup

Claude iOS voice mode is key for non-typing workflows:

1. Open Claude app
2. Tap microphone icon
3. Test: "Summarize the latest commits in databayt/hogwarts"
4. Verify response is clear and accurate

---

## Verification Checklist

- [ ] Claude app opens, signed in, voice works
- [ ] GitHub app shows databayt repos
- [ ] Safari autofills passwords from Keychain
- [ ] Notifications arrive from Claude, GitHub
- [ ] Can view and comment on GitHub issues from phone
