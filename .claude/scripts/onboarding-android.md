# Computer Onboarding — Android

> Step-by-step checklist for setting up a new Android phone for databayt work.
> Time: ~10 minutes.

## Prerequisites

- Google account signed in
- Wi-Fi connected
- Chrome installed (usually pre-installed)

---

## Apps to Install (Play Store)

Open Play Store and install each:

- [ ] **Claude** — AI assistant (search "Claude by Anthropic") *or* use claude.ai in Chrome
- [ ] **GitHub Mobile** — repos, issues, PRs, code review
- [ ] **Slack** — team communication (if workspace active)
- [ ] **Chrome** — already installed, configure below

### Optional

- [ ] **Outlook** — company email (hi@databayt.org)
- [ ] **Google Keep / Notes** — quick notes and handoffs

---

## Account Sign-ins

### Claude
Option A — **Claude App** (if available on Play Store):
1. Open Claude app
2. Sign in with Anthropic account
3. Verify conversations work

Option B — **claude.ai in Chrome** (always works):
1. Open Chrome
2. Navigate to `claude.ai`
3. Sign in with Anthropic account
4. Add to Home Screen: Chrome menu (three dots) > "Add to Home screen"
5. Now it works like an app

### GitHub Mobile
1. Open GitHub app
2. Sign in with GitHub account
3. Go to databayt org — verify you see repos (hogwarts, kun, etc.)
4. Enable notifications for mentions and review requests

### Slack (if active)
1. Open Slack
2. Join workspace: use invite link from team lead
3. Join channels: #general, #dev, #qa

---

## Chrome Configuration

1. **Settings > Passwords** — enable "Save passwords" and "Auto Sign-in"
2. **Settings > Autofill** — enable for addresses and payment methods
3. Sign in to Chrome with your Google account for password sync
4. Test: open `github.com` — should autofill login

---

## Notifications

1. **Claude / claude.ai** — allow notifications
2. **GitHub** — allow notifications (PR reviews, issue mentions)
3. **Slack** — allow notifications (direct messages, mentions only)

---

## Voice Input Setup

Android has built-in voice typing:

1. **Settings > System > Language & Input > On-screen keyboard**
2. Enable **Google Voice Typing** (Gboard)
3. Use microphone icon on keyboard in any app
4. Test in Claude: tap mic, ask "What are the open issues in databayt/hogwarts?"

---

## Verification Checklist

- [ ] Claude accessible (app or PWA), signed in
- [ ] GitHub app shows databayt repos
- [ ] Chrome autofills passwords
- [ ] Notifications arrive from Claude, GitHub
- [ ] Can view and comment on GitHub issues from phone
- [ ] Voice input works in Claude
