# Project Brief: Kun (كن)

> **Status**: Active

---

## 1. What is Kun

Kun (كن — Arabic for *Be*) is **the configuration** — agents, skills, hooks, MCP servers, rules, and memory — that makes Claude Code, Claude Desktop, Cowork, and the Agent SDK work together as a single engine running both technical and business sides of a software organization.

Kun is not a server. Not a platform. It is the brain that coordinates repositories, roles, and every tool in the stack.

**One word, and it happens.**

---

## 2. The Engine

| Component | Purpose |
|-----------|---------|
| **Agents** | Specialized AI across business, product, and tech chains |
| **Skills** | One-command workflows |
| **MCP Servers** | External tool integrations |
| **Rules** | Auto-activating guardrails on file patterns |
| **Hooks** | Guaranteed automation on lifecycle events |
| **Keywords** | Single words that map to complete workflows |

Live counts come from `.claude/memory/kun-inventory.json` via `bash scripts/inventory.sh`.

---

## 3. How It Works

| You Say | What Happens |
|---------|--------------|
| `dev` | Kill port 3000 → start server → open browser |
| `deploy` | Build → push to Vercel → retry until live |
| `saas billing` | Schema → actions → UI → payment integration |
| `handover auth` | Multi-pass QA across environments |

---

## 4. Three Phases

### Phase 1: Developer Engine (done)

CLAUDE.md hierarchy, agent fleet, skills, MCP servers, rules, hooks, memory, keyword mappings — all operational.

### Phase 2: Team Engine (current)

Role-based onboarding, shared settings via git, Agent Teams, CI/CD, business workflows.

### Phase 3: Company Engine (future)

Agent SDK pipelines, cost optimization, multi-customer scaling, pattern distribution.

---

## 5. Philosophy

- **Configuration over infrastructure** — don't build what Anthropic ships
- **Architecture over vibe coding** — humans design, AI generates within constraints
- **Patterns as training data** — every good component teaches the next
- **Open source** — build in the open, sustain through value
