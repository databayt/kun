# kun-content

Role profile for content + R&D work. Tuned for Samia.

## Effective surface

Adds to business profile:
- `/figma` — design ↔ code
- `/translate` — full-coverage translation sweep
- `/motion` — Framer Motion animations
- `/learn`, `/analyze`, `/pattern` — research surface

Denies same as business: no `/deploy`, `/ship`, engineering pipeline.

## Install

```bash
claude --plugin-dir /path/to/kun/plugins/kun-core \
       --plugin-dir /path/to/kun/plugins/kun-captain \
       --plugin-dir /path/to/kun/plugins/kun-content
```
