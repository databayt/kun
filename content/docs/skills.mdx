# Skills

Slash commands that trigger complete workflows. Say a keyword or type `/command` — the engine handles the rest.

## Workflow

| Skill | Trigger | What It Does |
|-------|---------|--------------|
| `/dev` | `dev` | Kill port 3000, start `pnpm dev`, open browser |
| `/build` | `build` | `pnpm build` + TypeScript check + auto-fix |
| `/quick` | `push` | Lint, fix, commit, push |
| `/deploy` | `deploy`, `ship` | Vercel deploy with retry |

## How Skills Work

Skills are SKILL.md files in `~/.claude/skills/<name>/` that expand into full prompts when invoked:

```bash
/proposal "Acme Corp"
```

The skill loads its template, invokes relevant agents and MCP servers, and produces output.
