# Kun (كن)

> The optimal configuration that makes Claude Code, Claude Desktop, Cowork, and the Agent SDK work together as a single engine running both the technical and business sides of **Databayt**.

## What is Kun?

Kun is not a server. Not a platform. It is the brain that coordinates **14 repositories**, **4 team members**, and every tool in the stack — from git commit to Stripe checkout to team standup.

**One word, and it happens.**

## The Company

**Databayt** (داتابايت) builds products that automate real-world operations — schools, rentals, commerce, healthcare — using a shared component library and a unified configuration engine.
