# CRM

Databayt runs a self-hosted CRM (cloned and deployed from [databayt/crm](https://github.com/databayt/crm)) with **one workspace per product**. Every team member has **one login that works on all workspaces** — log in once on any link, then move between them with the **workspace switcher (top-left)**.

## Login

| Property | Value |
| -------- | ----- |
| **Username** | `<name>@databayt.org` — `abdout`, `aseel`, `sedon`, `moutaz`, `ibrahim`, `samia`, `moed` |
| **Scope** | Same login on every workspace |
| **Switcher** | Top-left, after signing in anywhere |

> **Security** — accounts were provisioned with a shared bootstrap password. **Change your password on first login**: a shared credential across 7 people and 4 workspaces is a real risk once client data (school pipelines, contacts) lands in the CRM. Rotation is each member's first task.
