# Dispatch

Async communication via Apple Notes. Three notes in a `Dispatch` folder, synced via iCloud across all Apple devices.

## Three Channels

| Note | Direction | Purpose |
|------|-----------|---------|
| **Dispatch/Captain** | Captain → operator | Updates, decisions, weekly summaries |
| **Dispatch/Cowork** | Cowork ↔ Code | Bridge between thinking and doing |
| **Dispatch/Inbox** | Operator → Captain | Instructions, approvals, priorities |

## Usage

### Write
```bash
dispatch.sh captain "Weekly review complete. Pilot on track."
dispatch.sh captain "Need approval: add Pro seat" decision
dispatch.sh cowork "Architecture planned. Issues created: #10, #11, #12"
```

### Read
```bash
dispatch.sh read inbox
dispatch.sh read cowork
dispatch.sh read captain
```

### Priority Levels

| Level | Prefix | When |
|-------|--------|------|
| `fyi` | Blue | Informational |
| `normal` | Clipboard | Standard update |
| `decision` | Yellow | Needs operator approval |
| `urgent` | Red | Immediate attention |

## Session Start Protocol

1. `dispatch.sh read inbox`
2. `dispatch.sh read cowork`
3. `gh issue list --state open`
4. Proceed with highest priority

## iCloud Sync

Notes sync via iCloud across devices. Read and write from anywhere.
