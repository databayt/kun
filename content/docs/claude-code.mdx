Setup guide for the Kun engine configuration via Claude Code.

## One-Liner Setup

### macOS / Linux

```bash
curl -fsSL https://raw.githubusercontent.com/databayt/kun/main/.claude/scripts/install.sh | bash
~/.claude/scripts/secrets.sh <YOUR_GIST_ID>
```

### Windows (PowerShell)

```powershell
irm https://raw.githubusercontent.com/databayt/kun/main/.claude/scripts/install.ps1 | iex
& "$env:USERPROFILE\.claude\scripts\secrets.ps1" -GistId <YOUR_GIST_ID>
```

This installs the Claude Code CLI, downloads engine configuration, and sets up secrets.

## What Gets Installed

| Component | Description |
|-----------|-------------|
| **Agents** | Architecture, framework, product reference agents |
| **Commands** | `/repos`, `/atom`, `/template`, `/block`, `/build`, `/deploy`, etc. |
| **Memory** | atoms, templates, blocks, reports, repositories |
| **MCP Servers** | GitHub, Neon, shadcn, browser, Vercel, Stripe, etc. |
| **Scripts** | install, sync, setup-secrets, sync-repos |

## Step-by-Step Installation

### Step 1: Install the CLI

```bash
# macOS / Linux
curl -fsSL https://claude.ai/install.sh | sh

# Windows (PowerShell)
irm https://claude.ai/install.ps1 | iex

# Verify
claude --version
```

### Step 2: Download Engine Config

```bash
# macOS / Linux
curl -fsSL https://raw.githubusercontent.com/databayt/kun/main/.claude/scripts/install.sh | bash

# Windows
irm https://raw.githubusercontent.com/databayt/kun/main/.claude/scripts/install.ps1 | iex
```

### Step 3: Set Up Secrets

```bash
# macOS / Linux
~/.claude/scripts/secrets.sh <YOUR_GIST_ID>

# Windows
& "$env:USERPROFILE\.claude\scripts\secrets.ps1" -GistId <YOUR_GIST_ID>
```

### Step 4: Restart and Run

```bash
claude
# or with alias
c
```

## Directory Structure

```
~/.claude/                      # Windows: %USERPROFILE%\.claude\
├── CLAUDE.md                   # Global instructions
├── CLAUDE.local.md             # Personal overrides (not synced)
├── settings.json               # Environment & hooks
├── mcp.json                    # MCP server configurations
├── .env                        # Secrets (auto-generated, gitignored)
├── agents/                     # Custom agents
├── skills/                     # Skill definitions (SKILL.md)
├── commands/                   # Legacy slash commands
├── rules/                      # Path-scoped rules
├── memory/                     # Persistent registries
└── scripts/                    # Setup scripts
```

## Shell Configuration

### macOS / Linux

Add to `~/.zshrc`:

```bash
export PATH="$HOME/.local/bin:$HOME/.claude/bin:$PATH"
alias c='claude --dangerously-skip-permissions'

# Auto-load secrets
[ -f "$HOME/.claude/.env" ] && export $(grep -v "^#" "$HOME/.claude/.env" | xargs)
```

### Windows (PowerShell Profile)

```powershell
Set-Alias -Name c -Value claude
function cc { claude --dangerously-skip-permissions $args }
```

## Secrets Management

Stored in a private GitHub gist and auto-loaded via setup scripts.

### Required
- `GITHUB_PERSONAL_ACCESS_TOKEN` — GitHub API access

### Per-Project Databases
- `DATABASE_URL_<PROJECT>` — separate URL per project DB

### AI Keys
- `ANTHROPIC_API_KEY` — Claude API
- `GROQ_API_KEY` — Groq API (optional)

### Auth
- OAuth credentials (Google, Facebook)
- `AUTH_SECRET` — NextAuth secret

### Services
- Stripe, Resend, Cloudinary, ImageKit, Telegram (as needed)

## Repository Sync

```bash
~/.claude/scripts/sync-repos.sh
~/.claude/scripts/sync-repos.sh status
~/.claude/scripts/sync-repos.sh <repo-name>
```

### Monitored Repos

- **Core libraries**: `codebase`, `shadcn`, `radix`, `kun`
- **Products**: `hogwarts`, `souq`, `mkan`, `shifa`
- **Specialized**: `swift-app`, `distributed-computer`, `marketing`

## Configuration Sync

```bash
# macOS / Linux
~/.claude/scripts/sync.sh

# Windows
& "$env:USERPROFILE\.claude\scripts\sync.ps1"
```

Updates CLAUDE.md, agents, skills, memory, and scripts. Your `CLAUDE.local.md` is preserved.

## Local Customizations

Create `~/.claude/CLAUDE.local.md` for personal overrides:

```markdown
# My Local Settings

## Personal Keywords
| Keyword | Action |
|---------|--------|
| `myproject` | cd to my main project |
```

Never overwritten by sync.

## Troubleshooting

### Windows: `claude` not found
```powershell
$env:Path = [System.Environment]::GetEnvironmentVariable("Path","Machine") + ";" + [System.Environment]::GetEnvironmentVariable("Path","User")
```

### macOS: Permission denied
```bash
chmod +x ~/.claude/scripts/*.sh
```

### MCP servers not loading
1. Check `~/.claude/mcp.json` exists
2. Verify environment variables are set
3. Restart Claude

## Permission Flags

| Flag | Purpose |
|------|---------|
| `--dangerously-skip-permissions` | Skip all prompts |
| `--allowedTools "Bash,Read,Write"` | Whitelist tools |
| `-p "prompt"` | Direct prompt mode |
| `-r` | Resume last session |
