## UI Pipeline

```
radix (accessible primitives)
  → shadcn (styled components)
    → codebase (atoms + templates)
      → products (hogwarts, mkan, souq, shifa)
```

| Layer | Repo | Description |
|-------|------|-------------|
| Primitives | `databayt/radix` | Unstyled, accessible (Dialog, Popover) |
| Components | `databayt/shadcn` | Styled with Tailwind |
| Atoms | `databayt/codebase` | 2+ primitives composed |
| Templates | `databayt/codebase` | Full-page layouts |

Both `radix` and `shadcn` are forks customized for Arabic RTL support.

## Upgrade Path

```
1. radix (primitives — upstream sync)
2. shadcn (components — upstream + RTL patches)
3. codebase (atoms/templates — test compatibility)
4. kun (configuration — update version refs)
5. products (hogwarts → mkan → shifa → souq)
```
