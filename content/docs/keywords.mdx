## How Keywords Work

When you type a keyword, the engine runs three steps:

1. **Recognition** — the word activates the corresponding agent or agent chain
2. **Channeling** — relevant MCP servers open connections to external services
3. **Manifestation** — the workflow runs: files written, servers started, code transformed

Some keywords are simple — one word, one effect. Others are compound:
- `test` runs your test suite
- `test login` targets the login flow specifically
- `saas billing` produces an entire billing feature: schema + API + UI + payment integration
